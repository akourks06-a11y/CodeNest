## How to Use

### In Admin Panel:
1. Go to Admin Dashboard
2. Click "Add Language" or "Edit Language"
3. Copy the SVG code for your desired language
4. Paste it into the "Icon (SVG Code)" field
5. Save!

### Adding Custom Icons:
You can find more SVG icons from:
- **Feather Icons**: https://feathericons.com/
- **Heroicons**: https://heroicons.com/
- **Lucide**: https://lucide.dev/
- **Simple Icons**: https://simpleicons.org/ (for brand logos)

### Icon Requirements:
- Must be valid SVG code
- Should include `viewBox`, `fill`, `stroke`, and `stroke-width` attributes
- Recommended size: 24x24 viewBox
- Use `stroke="currentColor"` to make icons adapt to theme colors

---

## Tips for Best Results

1. **Keep it Simple**: Simple, clean icons look more professional
2. **Consistent Style**: All icons should have similar stroke width and style
3. **Test in Light/Dark**: Make sure icons are visible in both themes (use `stroke="currentColor"`)
4. **Size Matters**: Icons should be legible at small sizes (24px)
5. **No Fixed Colors**: Avoid hardcoded colors like `stroke="#ff0000"`, use `currentColor` instead

```html
<svg viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2"><path d="M12 2c-2.5 0-4.5 2-4.5 4.5v1c0 .8.7 1.5 1.5 1.5h6c.8 0 1.5.7 1.5 1.5v1c0 2.5-2 4.5-4.5 4.5s-4.5 2-4.5 4.5v1c0 .8.7 1.5 1.5 1.5h6c.8 0 1.5-.7 1.5-1.5v-1c0-2.5 2-4.5 4.5-4.5"/><circle cx="15" cy="6" r="1"/><circle cx="9" cy="18" r="1"/></svg>
```

2. In Admin Panel → Languages → Add Language
3. Paste the SVG code in the "Icon" field
4. Fill in Name: "Python"
5. Fill in Description: "Learn versatile programming with Python"
6. Save!
